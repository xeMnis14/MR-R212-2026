## Points clés

### 1. [Le modèle événement -> état -> rendu]

événement : clique sur la barre de recherche
état : proposition de recherche: True -> False
Rendu : classe .is-open ajouté, les propositions apparaissent 

### 2. [classList.toggle]

ça permet de faire le lien entre JavaScript et CSS pour regarder si la classe concerné ou non existe pour pouvoir la supprimer et l'afficher en cliquant sur un bouton par exemple, faire apparaitre un menu, ce côté interactif est lié au JS et donc afficher le menu est lié au CSS en passant par cette fonction.
